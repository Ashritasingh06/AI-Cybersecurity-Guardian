\# AI Personal Cybersecurity Guardian



AI Personal Cybersecurity Guardian is an AI-powered system designed to help users identify potential cybersecurity threats such as phishing emails and suspicious links.



\## Features

\- Detect phishing messages

\- Identify suspicious keywords

\- Provide security warnings

\- Suggest safe actions



\## Technologies Used

\- Python

\- Machine Learning

\- Natural Language Processing

\- Streamlit
